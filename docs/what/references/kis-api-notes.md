## 2. 토큰 관리

- 접근토큰 유효기간: **24시간**
- 만료 30분 전 자동 갱신 (TokenManager)
- 토큰 만료 에러 코드: `EGW00123` → 재발급 후 재시도
- Rate limit 에러 코드: `EGW00201` → 1초 대기 후 재시도
- 토큰 파일 캐싱: `./config/kis_token.json`

---

## 3. WebSocket 구독 제한

| 항목 | 제한 |
|------|------|
| 세션당 실시간 체결(H0STCNT0) + 호가(H0STASP0) 합산 | 20개 (향후 60개 예정) |
| 체결통보(H0STCNI0) | 별도 1개 |

**2-Tier 구독 모델:**
- Tier 1 (WebSocket): IN_POSITION + ENTRY_TRIGGERED + WATCHING 상위 20개
- Tier 2 (REST 폴링 3초): 나머지 관심종목 최대 30개

## 6. 주의사항

- 모의투자에서 **IOC/FOK/조건부 주문 불가** (지정가·시장가만 허용)
- 시장가 주문 시 `ORD_UNPR = "0"` 필수
- 호가단위 불일치 시 API 거부 (`APBK0634`) → 자동 보정 후 재시도
- 장 마감 3분 전부터 신규 주문 차단 권장
- 동시호가 시간(08:30~09:00, 15:20~15:30)에 시장가 주문 차단
